# busbra-augmentation-benchmark
Code and results for the paper "Systematic Evaluation of Data Augmentation Strategies for Ordinal BI-RADS Classification in Breast Ultrasound: A Benchmark on BUS-BRA" (AII 2026). Includes training pipeline, six augmentation strategies, EfficientNet-B3 and ResNet-50 backbones, and 5-fold cross-validation results on the BUS-BRA dataset.
